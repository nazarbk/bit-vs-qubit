## 🚀 About the Project

This project was built with the goal of making quantum physics accessible through web interactivity. Instead of reading long academic texts, users can interact with simulated "qubits", observe how wave functions collapse, and hack a communication channel in real-time.

It is designed as a 6-module progressive experience, making it an ideal showcase for advanced UI/UX skills, React state management, and complex fluid animations.

---

## 🧠 Interactive Modules

* **Module 1: The Qubit** - Introduction to the basic unit of quantum information.
* **Module 2: Superposition** - Visualization of how a particle can exist in multiple states at once.
* **Module 3: Measurement** - Simulation of the wave function collapse when observing a quantum system.
* **Module 4: No-Cloning Theorem** - Demonstration of the impossibility of copying unknown quantum states.
* **Module 5: Entanglement** - The phenomenon of "spooky action at a distance" connecting two particles.
* **Module 6: Cryptography (BB84)** - A minigame to intercept a quantum channel and see how physics exposes the spy.

---

## 🛠️ Tech Stack

* **Frontend Framework:** React 18 (Vite)
* **Animations:** Framer Motion (Fluid transitions, micro-interactions, and physics simulations)
* **Mathematics:** React-KaTeX (LaTeX formula rendering)
* **Icons:** Lucide-React
* **Styling:** Native CSS3 (CSS Variables, Flexbox, Grid, Glassmorphism)
* **Internationalization (i18n):** Custom Context API (EN/ES support)
